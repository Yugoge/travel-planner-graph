# Example: Inter-City Route Planning

> **See Complete Example**: commands/gaode-maps/examples/inter-city-route.md

## Quick Overview

Demonstrates planning multi-city travel using transit routing.

### Scenario
Plan transportation between major Chinese cities with budget and time constraints.

### Tools Used
- `transit_route` for inter-city trains
- `poi_search` for accommodations/attractions

### Key Steps
1. Research transit options (high-speed rail, regular trains, buses)
2. Compare schedules, costs, duration
3. Make recommendation based on user preferences
4. Structure itinerary with departure/arrival times
