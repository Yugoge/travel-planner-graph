## Next Steps

### User Actions Required

1. **Install MCP Server**:
   ```bash
   npm install -g rednote-mcp
   ```

2. **Authenticate with RedNote**:
   ```bash
   rednote-mcp init
   ```
   - Follow interactive login prompts
   - Cookie will be saved to `~/.mcp/rednote/cookies.json`

3. **Configure Claude Desktop** (if using Claude Desktop app):
   Add to `~/.config/Claude/claude_desktop_config.json`:
   ```json
   {
     "mcpServers": {
       "rednote": {
         "command": "rednote-mcp",
         "args": ["--stdio"]
       }
     }
   }
   ```
